# Guess Quest
An aggregate of 3 guessing games: 
- Number Guessing
- Hangman
- Mystery Code. 

## Features
- Various difficulties
- Score tracking and Score balancing
- Profile management 
- Leaderboard

## How to use
- Various difficulties
- Score tracking and Score balancing
- Profile management 
- Leaderboard

## Build
You will need a C compiler such as GCC (via [MinGW](https://www.mingw-w64.org/)) or Visual Studio Build Tools.
Compile the program:
```
gcc GamesCollection.c -o GuessQuest
```
